

## docker maken en starten

- start je docker desktop op

- ga nu in je commandprompt/terminal naar de directory `M5ProgDocker`
    > dus waar `docker-compose.yml` in staat
    
    > ![](img/mappen.PNG)


- type:
    - docker compose up -d
    - wacht tot alles klaar is:
        >![](img/dockerdone.PNG)
